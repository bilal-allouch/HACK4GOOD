Prompting pro

Astuces secrètes & efficaces

Les techniques que les experts utilisent pour obtenir des résultats 10× meilleurs.

01
Structure

Donnez un rôle précis

Commencez toujours par Tu es un expert en.... Définir un persona améliore la qualité des réponses jusqu'à 40%.

02
Contrôle

Contraignez le format de sortie

Spécifiez explicitement maximum 3 bullet points ou réponds en JSON. La précision réduit les tokens inutiles.

03
Raisonnement

Activez le Chain-of-Thought

Ajoutez Réfléchis étape par étape pour les tâches complexes. Améliore drastiquement la fiabilité sur les problèmes logiques.

04
Économie

Compressez votre contexte

Ajoutez Sois concis. en fin de prompt pour économiser 20-30% de tokens en sortie sur de gros volumes.

05
Précision

Exemples few-shot

Fournissez 2-3 exemples de ce que vous attendez. Le modèle imite les patterns — technique la plus efficace pour formater les sorties.

06
Avancé

Le délimiteur magique

Isolez le contenu variable avec --- ou <<<texte>>> pour éviter les injections accidentelles.
